代码高亮

get3w 已经支持超过 100 种语言代码高亮显示，在此感谢 Pygments。要使用 Pygments ，你必须安装 Python 并且在配置文件中设置 highlighter 为 pygments。

另外，你也可以使用 Rouge 实现代码高亮。虽然它不像 Pygments 支持那么多语言，但也可以胜任大多数场景。而且，由于 Rouge 基于 Ruby ，你无须在系统中配置 Python 环境。
